# Tweet_Plant

**Iniciamos este repositorío con la idea de implementar una planta que sea que sea capaz de tener un perfíl social.**

## Materiales 
- Raspberry PI
- Cámara
- Sensor de Humedad
- Sensor Galga
- Sensor de Luz
- Sensor de Temperatura
- Sensor de Presión
- Cableado (estaño)
